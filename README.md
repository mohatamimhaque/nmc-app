# National Mathematics Carnival 2026 Website

Official event website and admin panel for National Mathematics Carnival 2026, built with Next.js and Supabase.

## PRD

See the full product requirements document: [nmc prd.md](nmc%20prd.md).

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Cloudflare R2 (media storage)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run start` - start production server
- `npm run lint` - run ESLint
